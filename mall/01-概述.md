```
1.学生管理系统、工资管理系统弊端
2.商品、订单、用户模块
3.知识点丰富
4.热点、趋势
```

## 2.	项目亮点

```
1.最新的业界的互联网企业级的优质技术
2.代码规范简洁，充分优化
3.流程完整，电商功能丰富
4.前后端彻底分离，符合未来趋势，学了就能用

```
