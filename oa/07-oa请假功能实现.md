# 请假功能实现

## 1.请假流程数据库设计

### 1.1工作流程表设计

```
请假单表				审批任务流程表				消息通知表
LeaveForm		--->	ProcessFlow		<---		Notice

表结构见02-oa数据库设计.md中的2.5、2.6、2.7
```

### 1.2设计约束

```
1.每一个请假单对应一个审批流程
2.请假单创建后，按业务规则生成部门经理、总经理审批任务
3.审批任务的经办人只能审批自己辖区内的请假申请
4.所有审批任务“通过”，代表请假已经批准
5.任意审批任务“驳回”操作后，其余审批任务取消，请假申请被驳回
6.请假流程中任意节点产生的操作都要生成对应的系统通知
```
